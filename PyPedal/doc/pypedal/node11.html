1.2 Where to get information and code

PyPedal and its documentation are available at: http://pypedal.sourceforge.net/. The Sourceforge site, http://sourceforge.net/projects/pypedal/, provides tools for reporting bugs (https://sourceforge.net/tracker/?func=add&group_id=106679&atid=645233, making feature requests (https://sourceforge.net/tracker/?func=add&group_id=106679&atid=645236), and discussing PyPedal (https://sourceforge.net/forum/?group_id=106679).
