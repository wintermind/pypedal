2.2 Testing the Python installation

The first step is to install Python 2.4 (or later) if you haven't already done so. Python is available at http://sourceforge.net/projects/python/. Click on the link corresponding to your platform, and follow the instructions presented there. Python can usually be started by typing "python" at the shell (Posix) or double-clicking on the Python interpreter (Windows). When you start Python you should see a message like: 
Python 2.5.2 (r252:60911, Apr  8 2008, 21:49:41)
[GCC 4.2.3 (Ubuntu 4.2.3-2ubuntu7)] on linux2
Type "help", "copyright", "credits" or "license" for more information.
If you have problems getting Python to work, contact your local support person or e-mail python-help@python.org for help.
